# Student Performance Prediction System

## 📌 Project Overview

The Student Performance Prediction System is a Machine Learning project that predicts student academic performance based on various educational and behavioral factors. The system analyzes student data and classifies performance into three categories:

* **H** – High Performance
* **M** – Medium Performance
* **L** – Low Performance

The project compares multiple machine learning algorithms and evaluates their prediction accuracy.

---

## 🚀 Features

* Data preprocessing and feature engineering
* Correlation heatmap visualization
* Interactive graphical analysis using Seaborn and Matplotlib
* Performance prediction using multiple machine learning models
* Comparison of model accuracy and classification metrics
* User input-based prediction system

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

---

## 🤖 Machine Learning Models

The project implements the following algorithms:

1. Decision Tree Classifier
2. Random Forest Classifier
3. Logistic Regression
4. Perceptron
5. Neural Network (MLP Classifier)

---

## 📊 Dataset

The dataset contains information related to student academic activities and engagement, including:

* Raised Hands
* Visited Resources
* Discussion Participation
* Student Absence Days
* Gender
* Nationality
* Grade Information
* Parent Survey Responses

These attributes are used to predict student performance levels.

---

## 📈 Results

The project evaluates different machine learning models using:

* Accuracy Score
* Precision
* Recall
* F1-Score
* Classification Report

Among the tested models, Logistic Regression and Random Forest achieved the best performance on the dataset.

## ▶️ How to Run

### Clone Repository

```bash
git clone https://github.com/Athithya-Ajay/Student-Performance-Prediction-System.git
cd Student-Performance-Prediction-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Project

```bash
python Project.py
```
